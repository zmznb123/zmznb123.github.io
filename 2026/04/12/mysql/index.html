mysql

数据类型：

数值类型

  • INT(年龄，id，数量)
  • TINYINT（状态值（0/1），性别）
  • BIGINT（超级大id）
  • DECLMAL（金额，价格）（10,2）【总共十位数保留两位小数】

字符串类型

  • varchar（N）（手机号，姓名，地址）
  • char(N)(固定长度，不够的补空格，短字符串（如身份证号18位，定长的编码）)
  • text（长文本，文章内容，评论等）

时间类型

  • datettime(格式：‘YYY-MM-DD HH:MM:SS’记录创建时间更新时间)
  • DATE（’YYY-MM-DD’只存日期，不存时分秒）
  • timestamp(时效性较强的数据（自动转换时区）)

1.数据库的连接

本地连接

  • mysql -u root -p

远程连接

  • mysql -h 地址 -p 端口 -u 用户名 -p

2.数据库操作

查看所有库

show databases

创建库

create database 库名；

使用库

use 库名；

删除库

drop database 库名；

3.数据表操作

查看所有表

show tables；

查看表结构

desc 表名；

创建表

create table 表名（

id INT PRIMARY KEY AUTO_INCREMENT,

字段名 数据类型 约束）；

4.增 insert

insert into 表名（字段1，字段2）VALUES（值1，值2）；

增加表值

  • alter table 表名 add column 数据名称 +数据类型 +限制[not null(不允许为空)，default‘未填写’（默认），unique（不能重复）]

5.查select

查询全部

SELECT * FROM 表名;

条件查询

SELECT * FROM 表名 WHERE 条件;

模糊查询

SELECT * FROM 表名 WHERE 字段 LIKE ‘%值%’;

排序

SELECT * FROM 表名 ORDER BY 字段 DESC;

分页

SELECT * FROM 表名 LIMIT 0,10;

6.改update

UPDATE 表名 SET 字段=值 WHERE 条件;

7.删delete

DELETE FROM 表名 WHERE 条件;

8.常用聚合函数

COUNT(*) 统计
SUM(字段) 求和
AVG(字段) 平均
MAX(字段) 最大
MIN(字段) 最小

9.进阶

分组

SELECT 字段 FROM 表名 GROUP BY 字段;

内连接

SELECT * FROM 表1
JOIN 表2 ON 表1.id = 表2.id;

– 插入数据（加了email字段）
INSERT INTO users (username,password,email)
VALUES (‘newuser’,’123456’,‘test@qq.com‘);

  • 查询数据
    SELECT * FROM users WHERE username=’admin’;

  • 修改数据
    UPDATE users SET password=’newpass’ WHERE username=’admin’;

  • 删除数据
    DELETE FROM users WHERE username=’test’;
