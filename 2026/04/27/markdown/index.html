Markdown 语法基础

一、标题

用 # 表示标题，# 数量代表标题级别，最多支持 6 级：

markdown

效果：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

二、文本格式

表格

效果 语法 示例
加粗 **内容** 或 __内容__ 加粗文本
斜体 *内容* 或 _内容_ 斜体文本
加粗 + 斜体 ***内容*** 加粗斜体
删除线 ~~内容~~ 删除线文本
行内代码 内容 行内代码

三、列表

1. 无序列表

用 - * + 任意一种都可以，后面加空格：

markdown

效果：

  • 苹果
  • 香蕉
  • 橙子

2. 有序列表

用 数字. 表示，后面加空格，数字不影响排序（推荐按顺序写）：

markdown

效果：

  1. 起床
  2. 洗漱
  3. 吃早餐

3. 嵌套列表

子列表前面加 2-4 个空格：

markdown

效果：

  • 水果

    • 苹果
    • 香蕉

  • 蔬菜

    1. 白菜
    2. 萝卜

1. 超链接

格式：[显示文本](链接地址 "可选标题")

markdown

效果：百度

2. 图片

格式：![图片描述](图片地址 "可选标题")

markdown

markdown

六、代码块

1. 行内代码

用一对反引号 ``` 包裹：

markdown

效果：这是 行内代码 示例

2. 代码块

用三个反引号 ````` 包裹，开头可以指定语言（如 python sql），实现语法高亮：

七、表格

用 | 分隔列，- 分隔表头和内容，: 控制对齐方式：

markdown

效果：

表格

姓名 年龄 性别
张三 18 男
李四 20 女

  • :--:：居中
  • ---:：右对齐

八、分割线

用三个及以上的 - * _ 表示，前后最好空一行：

markdown

九、转义字符

如果想显示 Markdown 语法符号本身（比如 * # []），可以在前面加反斜杠 \ 转义：

markdown

效果：

* 这不是斜体，而是显示星号

# 这不是标题，而是显示井号
