一、PHP 基础入门

1. 基本语法

  • PHP 代码以 <?php 开始，?> 结束，可嵌入 HTML 中
  • 语句必须以分号 ; 结尾
  • 单行注释：// 注释内容
  • 多行注释：/* 注释内容 */

2. 输出语句

  • echo：最常用，输出一个或多个字符串，无返回值
  • print：输出一个字符串，有返回值（效率低于 echo）
  • print_r()：打印变量信息（适合数组 / 对象）
  • var_dump()：打印变量类型 + 值（调试专用）

二、变量与数据类型

1. 变量规则

  • 以 $ 符号开头，如 $name
  • 区分大小写，不能以数字开头
  • 弱类型语言：无需声明类型，自动转换

2. 8 种数据类型

标量类型（4 种）

  1. 字符串：$str = "abc";
  2. 整型：$num = 100;
  3. 浮点型：$float = 3.14;
  4. 布尔型：$bool = true / false;

复合类型（2 种）

  1. 数组：$arr = [1,2,3];
  2. 对象：面向对象使用

特殊类型（2 种）

  1. NULL：空值
  2. 资源：文件、数据库连接等

3. 常量

  • 定义后不可修改，全局有效
