九、核心注意事项（新手必看）

  1. Java 严格区分大小写
  2. 文件名必须和 public class 类名完全一致
  3. 所有符号必须是英文半角
  4. 字符串用双引号 ""，字符用单引号 ''
  5. 数组下标从 0 开始
  6. 判断字符串相等用 equals()，不要用 ==
  7. 循环和 switch 中别忘记 break
