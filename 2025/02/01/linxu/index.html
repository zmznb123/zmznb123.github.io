Linux基础操作

  • linux主要特征 ：一切皆文件（目录、硬盘等都是文件）；硬件都在/dev 目录，如硬盘、U盘为/dev/sd[a-d]； /dev/sr0（/dev/cdrom）是光驱的设备名（df命令查看），为设备文件，代表的是光驱本身，得把这个设备挂载到目录下（一般为/mnt）(文件系统的临时挂载点)，才能对设备上的文件进行读写等操作；
  • 不懂的命令 ：man 命令（查用法、全称），只记得关键词，可用man -k 关键词；

查看文件头部内容(head)

默认用法：head test.txt //查看文件前十行内容

-n NUM：指定显示的行数。

-c NUM：指定显示的字节数，而不是行数。

-q：不显示文件名。

-v：显示文件名。

查看文件尾部内容(tail)

默认用法：tail test.txt //查看文件后十行内容

-n NUM：指定显示的行数。

-c NUM：指定显示的字节数，而不是行数。

-f：实时跟踪文件（”follow”），用于查看文件末尾的新内容。这个选项通常用于监控日志文件。

-F：与 -f 类似，但会在文件被截断或重新创建时自动重新打开文件。

-q：不显示文件名（对于多个文件时）。

-v：显示文件名

编辑文件(vi、vim)

vim是vi的升级版

启动：vim 文件名

（如果有则打开，没有则新建）

普通模式（Normal Mode）：默认模式，用于导航、删除、复制、粘贴、查找等操作。

插入模式（Insert Mode）：用于编辑文本，类似于普通的文本编辑器。

命令模式（Command Mode）：用于输入命令，例如保存、退出、搜索等。

模式切换：

  • 从普通模式进入插入模式：按 i、I（光标前）、a、A（光标后）、o（光标下方新行）、O（光标上方新行）等。
  • 从插入模式返回普通模式：按 Esc 键。
  • 从普通模式进入命令模式：按 :（冒号）。

常见操作：

1. 基本导航（在普通模式下）：
  • h：左移一个字符。
  • j：下移一行。
  • k：上移一行。
  • l：右移一个字符。
  • w：跳到下一个单词的开头。
  • b：跳到前一个单词的开头。
  • 0：跳到当前行的开头。
  • $：跳到当前行的末尾。
  • gg：跳到文件的开头。
  • G：跳到文件的末尾。
  • Ctrl+f：向下翻一页。
  • Ctrl+b：向上翻一页
2. 编辑文本（在插入模式下）：
  • i：进入插入模式，在当前光标前插入。
  • I：进入插入模式，在行首插入。
  • a：进入插入模式，在当前光标后插入。
  • A：进入插入模式，在行尾插入。
  • o：进入插入模式，在当前行下插入新行。
  • O：进入插入模式，在当前行上插入新行。
  • Esc：退出插入模式，返回普通模式。
3. 删除、复制、粘贴（在普通模式下）：
  • x：删除当前光标所在的字符。
  • dd：删除当前行。
  • d + w：删除一个单词。
  • d + 0：删除从当前光标到行首的内容。
  • d + $：删除从当前光标到行尾的内容。
  • yy：复制当前行。
  • y + w：复制一个单词。
  • p：在光标后粘贴。
  • P：在光标前粘贴。
4. 撤销和重做：
  • u：撤销上一步操作。
  • Ctrl+r：重做撤销的操作。
5. 查找和替换（在命令模式下）：
  • /pattern：向下查找 pattern。
  • ?pattern：向上查找 pattern。
  • n：跳到下一个匹配。
  • N：跳到上一个匹配。
  • :%s/old/new/g：将文件中所有的 old 替换为 new（g 代表全局替换）。
  • :s/old/new/g：在当前行中替换所有的 old 为 new。
6. 保存和退出（在命令模式下）：
  • :w：保存文件。
  • :q：退出 vim。
  • :wq 或 ZZ：保存并退出 vim。
  • :q!：强制退出（不保存更改）。
  • :x：保存并退出（与 :wq 等效）。

用户和权限管理

whoami：显示当前登录的用户名

id：显示显示用户的 UID、GID 及所属的组

who：命令用于查看当前所有登录系统的用户信息。

passwd：修改用户密码

useradd：添加新用户

userdel：删除用户

usermod：修改用户信息

su：切换用户

groupadd：添加新组

groupmod：用户组编辑

groupdel：用户组删除

groups：查看用户所属的组

chmod：修改文件或目录的权限

注意：

如果想要给文档设置权限，操作者要么是root用户，要么就是文档的所有者。

chown：改变文件或目录的所有者
